我的Emacs配置

我的emacs配置，已经不再使用

https://github.com/hongxchen/emacs.d

现在使用prelude的配置

https://github.com/bbatsov/prelude.git

具体配置方法：

(1) 安装Emacs24

mac:
brew install emacs

linux:
sudo apt-get install emacs
sudo yum install emacs

(2) 安装第三方插件

mac:
brew install ctags global

linux:
sudo apt-get install ctags global clang
sudo yum install ctags global clang

(3) 安装prelude的配置

curl -L http://git.io/epre | sh

(4) 自定义配置文件

cd ~/.emacs.d/
vim prelude-modules.el

选择需要打开的mode，去除开头的;;,我的设置如下：
;;; Uncomment the modules you'd like to use and restart Prelude afterwards

;; Emacs IRC client
(require 'prelude-erc)
(require 'prelude-ido) ;; Super charges Emacs completion for C-x C-f and more
;; (require 'prelude-helm) ;; Interface for narrowing and search
(require 'prelude-company)
;; (require 'prelude-key-chord) ;; Binds useful features to key combinations
;; (require 'prelude-mediawiki)
;; (require 'prelude-evil)

;;; Programming languages support
(require 'prelude-c)
;; (require 'prelude-clojure)
;; (require 'prelude-coffee)
;; (require 'prelude-common-lisp)
;; (require 'prelude-css)
(require 'prelude-emacs-lisp)
(require 'prelude-erlang)
(require 'prelude-go)
;; (require 'prelude-haskell)
;; (require 'prelude-js)
;; (require 'prelude-latex)
(require 'prelude-lisp)
(require 'prelude-org) ;; Org-mode helps you keep TODO lists, notes and more
;; (require 'prelude-perl)
;; (require 'prelude-python)
;; (require 'prelude-ruby)
;; (require 'prelude-scala)
(require 'prelude-scheme)
(require 'prelude-shell)
;; (require 'prelude-scss)
;; (require 'prelude-web) ;; Emacs mode for web templates
(require 'prelude-xml)

(5) 安装mode

打开emacs，会自动下载mode所需的文件

(6) 重启emacs
